# App
Running the app:

Linux
- export FLASK_APP=fitnessbuddy
- export FLASK_ENV=development
- flask init-db
- flask run

Windows
- set FLASK_APP=fitnessbuddy
- set FLASK_ENV=development
- flask init-db
- flask run
